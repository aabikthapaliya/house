  • Do a proper vacuum — especially common areas, living area, and kitchen

  • Just ensure the cleaning is done nicely

🧽 Mopping

  • Mop all hard floors using the new mop
  • Use the old mop for toilets only
  • Both toilets need to be done

🛁 Bathrooms

  • If toilet paper is empty → restock it
  • Extra stock is kept in the garage
  • If running low on stock → let Aabi know
  • Bath mats need to be washed every week
  • Put them in washing machine for 45 minutes, then dryer
  • Make sure to set them back in place afterward

  • Range hood is handled by Aabi monthly — you don't need to worry about it
